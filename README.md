# NFL Combine Analytics Tool

An interactive web app that analyzes NFL Combine data using position-adjusted scoring.

## Features
- Athletic score (0–100)
- Position ranking
- Score tiers (Elite, Great, etc.)
- Player search
- Similar player matching
- Interactive dashboard (Streamlit)

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
